HTML IS AN HYPERTEXT MARKUP LANGUAGE WHICH NOT AN PROGRAMMING LANGUAGE
IT IS USED TO CREAT WEB PAGE DESIGNS
 IT CONSIST OF TAGS AND ATTRIBUTES 

 Text-related tags:
<h1> to <h6>: Headings (from largest to smallest).
<p>: Paragraph.
<a>: Anchor tag (used for links).
<strong>: Defines important text (bold).
<em>: Defines emphasized text (italic).

Media-related tags:
<img>: Embeds images.
<audio>: Embeds audio files.
<video>: Embeds video files.

Container and structure tags:
<div>: Defines a division or section (block-level).
<span>: Defines a small section of content (inline).
<section>: Represents a section of content.
<article>: Defines independent content (like an article).


<li>: List item.

Table tags:
<table>: Defines a table.
<tr>: Defines a table row.
<td>: Defines a table cell.
<th>: Defines a table header cell.

List tags:
